SDLGame

A game engine built with SDL2 and Box2D.

What is this???

SDLgame is a game engine built with SDL2 and Box2D. It is written in C++ with Visual Studio 2015. Fun fact: I'm using the Windows 8.1 SDK to build, and interestingly enough, the executable works on Windows 7!

Source?

Look above for the source tarball or zip package.

Compiling

Use VS2015, and make sure SDL2 is in c:\sdk\sdl2. As Box2D is implemented, make sure it is compiled and located at c:\sdk\box2d.
